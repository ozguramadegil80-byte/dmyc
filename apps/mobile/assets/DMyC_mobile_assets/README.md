# DMyC Mobile App Icon Assets

Bu paket, yüklenen DMyC logosundan mobil uygulama için hazırlanmış launcher/app icon dosyalarını içerir.

## Klasörler

- `ios/AppIcon.appiconset/` — Xcode içine direkt eklenebilir iOS AppIcon seti.
- `android/app/src/main/res/` — Android legacy ve adaptive launcher icon kaynakları.
- `expo-react-native/` — Expo / React Native için `icon.png`, `ios-icon.png`, `adaptive-icon.png`, `splash-icon.png`, `favicon.png`.
- `store/` — App Store 1024x1024 ve Google Play 512x512 görselleri.
- `web/` — favicon ve apple-touch-icon çıktıları.
- `preview/asset-preview.png` — farklı boyutlarda hızlı kontrol görseli.
- `source/` — düzenlenmiş kare kaynak kopyaları.

## Android adaptive background color

Kullanılan arka plan rengi: `#0B1015`

## Not

Kaynak logo 450x449 px olduğu için büyük mağaza ikonları 1024px'e kaliteli şekilde büyütüldü; gerçek vektör dosyası varsa daha keskin sonuç için aynı paket vektörden yeniden üretilebilir.
